# 🛍️ Fashion Design E-Commerce Store

## 📌 وصف المشروع
مشروع متجر إلكتروني مخصص لبيع ملفات رقمية تحتوي على **تصاميم أزياء**.  
المتجر موجّه للأفراد أو الشركات العاملة في مجال تصميم الأزياء، ويتيح لهم عرض وبيع منتجاتهم بشكل احترافي عبر الإنترنت.

---

## 🚀 المميزات
- إدارة المنتجات (رفع، تعديل، حذف).
- دعم بيع الملفات الرقمية (تصاميم أزياء بصيغة PDF أو صور).
- نظام تسجيل دخول وحماية باستخدام **JWT Authentication**.
- تكامل مع خدمات تخزين سحابي مثل **AWS S3** لرفع الملفات.
- واجهة برمجية (APIs) منظمة لتسهيل التكامل مع تطبيقات أخرى.

---

## 🛠️ التقنيات المستخدمة
- **ASP.NET Core** (Backend)
- **Entity Framework Core** (ORM)
- **SQL Server** (قاعدة بيانات)
- **JWT Authentication** (الحماية)
- **AWS S3** (تخزين الملفات)
- **Git & GitHub** (إدارة الكود)

---

## ⚠️ ملاحظات هامة
- تم استبدال جميع المفاتيح والقيم الحساسة بقيم وهمية لأغراض العرض فقط.  
- تم حذف أو قص بعض الأجزاء من المشروع، حيث أن النسخة المنشورة مخصصة للعرض والتوضيح وليست النسخة الكاملة القابلة للاستخدام المباشر.
